# test-todo-list

> :ledger: Example of TODO list

## Install

```html
<script src="scripts/todo-list-library.js"></script>
<script>
function setup() {
    new TestTODOList({
        $input: document.querySelector('....'),
        $list: document.querySelector('...')
    });
}

window.addEventListener('DOMContentLoaded', setup);
</script>
```
